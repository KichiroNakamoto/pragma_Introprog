Ejercicio #1

Consulta los otros tipos de meta y que significa cada uno dentro del documento HTML:

Meta de "metainformation" - metainformación. Sirve para aportar información sobre el documento. Las meta etiquetas (también llamadas: meta tags) se utilizan al crear páginas en HTML o XHTML. Estas etiquetas pueden describir el contenido de un sitio web con precisión. Este elemento te permite transmitir metadatos estructurados a través de la página web. Los metadatos pueden variar dependiendo de la descripción de los atributos deseados. Las meta etiquetas se pueden utilizar para describir una página con más detalle.

Existen muchas meta etiquetas distintas que se pueden incluir en la cabecera de una página web

  • meta charset

    • Esta meta etiqueta se denomina charset y define la codificación de caracteres que emplea el documento HTML. Si no se utiliza habrá caracteres que no serán legibles, como por ejemplo la ñ o las tildes, que no se mostrarán correctamente.

  • meta name

    • Estas meta tags indican quién es el autor de la página web y quién tiene los derechos sobre el código HTML. Por lo general estas etiquetas se generan automáticamente en las plataformas como WordPress. Las palabras clave (keywords) son aquellos términos que los usuarios introducen en los motores de búsqueda. Esta meta etiqueta se utiliza para definir las palabras clave que se quieren asociar a la página web

  • meta property

    • La etiqueta meta siempre está dentro de la etiqueta. Básicamente, puede usar el atributo de propiedad meta en la etiqueta del cuerpo, pero debe contener el elemento de propiedad. Este atributo se utiliza básicamente para explicar el nombre de la propiedad. Este atributo se usa básicamente para obtener un encabezado de mensaje de respuesta HTTP

Ejercicio #2

Explica el significado de estas líneas de código:

  • meta name="description" content="Ejemplo de HTML5"

    • En la descripción o meta description podemos describir brevemente el contenido de la página web. Esta información se muestra como una síntesis en dos líneas del tema de una página que aparece bajo la URL) en los buscadores de uso más generalizado como Google o Bing, por lo que se recomienda cuidar su redacción. Por su papel central en tanto que influye de manera decisiva en la elección del usuario del mejor resultado acorde con su búsqueda, este meta tag es considerado uno de los más importantes en cuanto a la optimización para los buscadores

  • meta name="keywords" content="HTML5, CSS3, JavaScript"

    • Las palabras clave meta son etiquetas meta que puede usar para brindar a los motores de búsqueda más información sobre el contenido de una página. Se encuentran en el código fuente HTML de una página web y no son visibles para los visitantes.

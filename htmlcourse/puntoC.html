Define: sitio web, pagina web, plataforma web

  • Sitio Web: Un sitio web es una colección de páginas web vinculadas (más sus recursos asociados) que comparten un único nombre de dominio
    Pagina Web: Hace referencia a una direccion un HTML por lo que un sitioweb esta compuesto por varias paginas.
    Plataforma Web: las aplicaciones web son plataformas principalmente interactivas que se centran en que los usuarios realicen acciones. Esta diferencia fundamental determina sus principales objetivos, características y funciones.
